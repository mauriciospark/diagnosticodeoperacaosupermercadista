Bloco 1: Onde o supermercado está hoje (Dados Gerais)

Explicação para o dono: "Aqui eu só preciso dos dados oficiais e do tamanho da operação. Isso me ajuda a entender a escala do projeto: atender 2 caixas é muito diferente de atender 10 caixas."

Bloco 2: Como a mercadoria se move (Logística e Estoque)

Explicação para o dono: "Quero entender como o produto sai do caminhão e chega na prateleira. O objetivo é descobrir se o senhor está perdendo dinheiro com produtos vencendo no estoque ou se falta mercadoria que o cliente quer comprar."

Bloco 3: A velocidade do seu caixa (Tecnologia e Operação)

Explicação para o dono: "Aqui é a parte técnica, focada na sua paz de espírito. Quero saber se o seu sistema atual trava, se ele é lento ou se as informações batem. Se o caixa é rápido, o cliente fica feliz e volta."

Bloco 4: O que o senhor precisa para decidir melhor (Visão de Gestão)

Explicação para o dono: "Por fim, quero saber quais informações o senhor gostaria de ter na mão agora e não tem. O projeto serve para transformar os dados de venda em ferramentas para o senhor tomar decisões mais seguras."
